# Pomodoro session notes

Stage 40 creates brief evidence-backed notes about the just-completed 30–60 minute work session.

- Reserve the next four-digit `POM` number in `../numbering-registry.md` before creating a note.
- Name notes `POM<NNNN>-<brief-kebab-description>.md`.
- Capture what just happened; do not turn a POM note into a timer, project summary, schedule, or future plan.
